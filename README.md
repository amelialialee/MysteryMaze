1. Introduction
-------------------------------------------------------------------------------
Welcome to The Planet Lyrnth Escape, a thrilling maze game that combines 
exploration, puzzle-solving, and strategic enemy avoidance.
Navigate through mazes, solve puzzles, collect items, and fight enemies to escape to leave the planet.
Are you ready for the escape?

-------------------------------------------------------------------------------
2. Features
-------------------------------------------------------------------------------
- Randomly Generated Mazes: Each level is unique, providing endless replayability.
- Maze Elements: Teleport, Break walls & Unlock doors to get to the goal faster
- Multiple Game Modes: Choose between different difficulty levels and Game Modes
- Puzzle-Solving Elements: Progress through the maze by solving various puzzles & sub-games
- Inventory Management: Decide whether to carry items forward as you progress, use items or discard them
- Progression & Leveling: Each level increases in difficulty, with more enemies, puzzles, and challenges...
- Save and Load Feature: Save your game progress and resume later.
- Leaderboard: Compete for the best times and highest levels in the Speed Run mode.
- Countdown: Want a challenge? Try the Countdown challenge mode and see how many levels you can Compete

-------------------------------------------------------------------------------
3. Game Controls
-------------------------------------------------------------------------------
- W, A, S, D: Move Up, Left, Down, Right
- H: View Help Menu
- Q: Quit the current game or return to the main dashboard
- Interact with objects such as doors, puzzles, and items 
- Other Inputs will be ask/ displayed accordingly

-------------------------------------------------------------------------------
4. Game Modes
-------------------------------------------------------------------------------
(1) DEFAULT (custom game): Finish the maze, complete all the levels
(2) COUNTDOWN CHALLENGE: Finish as many levels within the countdown
(3) PUZZLE COMPLETION: Finish all puzzles to be able to proceed to the next level
(4) GUARDIAN ELIMINATION: Collect Swords & fight off all guardians in the maze to proceed to the next level
(5) SPEED RUN RACE (to compete in the LEADERBOARD): Finish as many mazes as fast as you can & see where you stand on the LEADERBOARD

-------------------------------------------------------------------------------
5. Installation & How to Run the Game (Windows)
-------------------------------------------------------------------------------
Note: This game is currently Windows only.

1. Download the game file (MysteryMaze.exe)
2. Double-click MysteryMaze.exe to launch the game
-------------------------------------------------------------------------------
6. How to Save and Load the Game
-------------------------------------------------------------------------------
Games can only be saved for:
- Uncompleted games (Optional): You will be prompted option to save game when you are trying to quit game
- Completed games in Speed Run Race game mode

To resume a previous game:
> Go to menu 
> Enter '4' for ("LOAD GAME")
> Enter Game Number (Indicated the top left of game details)
